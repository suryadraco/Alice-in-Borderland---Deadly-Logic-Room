# Alice in Borderland - Deadly Logic Room

A survival puzzle game inspired by the hit Netflix series "Alice in Borderland". Test your logic and survival instincts across 100 increasingly difficult levels.

## Game Overview

**Four doors. Only ONE leads to survival.**

In this deadly logic room game, you must solve puzzles by choosing the correct door. Each level presents you with cryptic hints that must be read literally. Choose wrong, and it's GAME OVER.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd strike-build-play-main

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The game will be available at `http://localhost:8080`
